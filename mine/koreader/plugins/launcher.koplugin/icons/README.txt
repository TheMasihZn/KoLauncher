Custom icon for Launcher toolbar button

Place your icon file in this folder and name it one of the following:

- launcher.png (preferred)
- launcher.svg
- launcher.jpg

Recommended size: 48x48 or 64x64 pixels (PNG) with transparent background.

After copying the icon, restart KOReader to see it in the top bar.

If no custom icon is found, the plugin falls back to a built-in appbar icon (appbar.toolbox).
